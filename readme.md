## Installation

To run any of the projects, follow these steps:

1. Clone the repository:
    ```sh
    git clone https://github.com/Aniket-a14/JAVA.git
    ```
2. Compile the Java files:
    ```sh
    javac *.java
    ```
3. Run the files:
    ```sh
    java MainClass
    ```
